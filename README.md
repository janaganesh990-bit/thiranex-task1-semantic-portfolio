# Task 1 — HTML5 Semantic Structure & Accessibility Portfolio

This project was built as part of my Web Development Internship at Thiranex.

## Objective

Build a multi-page personal portfolio website using semantic HTML5 and WCAG accessibility guidelines.

## Features Implemented

* Semantic HTML5 structure (`header`, `nav`, `main`, `section`, `article`, `footer`)
* ARIA labels for screen readers
* SEO-friendly meta tags
* Accessible tab-navigable contact form
* Keyboard focus indicators
* Skip navigation link

## Technologies Used

* HTML5
* CSS3
* Accessibility (WCAG)

## Outcome

Designed a fully accessible portfolio skeleton targeting high Lighthouse Accessibility and SEO scores.
